🌦️ Weather App

📌 Project Description

This is a simple Weather App built using HTML, CSS, and JavaScript.
It fetches real-time weather data from the OpenWeatherMap API and displays weather details based on user input.


🚀 Features

- Search weather by city name
- Displays temperature in Celsius
- Shows weather condition (cloudy, sunny, etc.)
- Displays humidity and wind speed
- Error handling for invalid city


🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeatherMap API
  

⚙️ How It Works
1. User enters a city name
2. Clicks on the search button
3. App sends a request to the API
4. Receives data in JSON format
5. Parses and displays weather details on UI
   

🔧 Setup Instructions
1. Clone the repository
2. Open the project folder
3. Add your API key in "script.js"
4. Run "index.html" in browser
